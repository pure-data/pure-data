Pd Manual

This is the Manual for Pd (Pure Data), a patchable environment for audio analysis, synthesis and digital signal processing. Pd can also be extended via ecternals to include yet more computer music features, multimedia capabilities and more. The latest version of this page can be found at: http://msp.ucsd.edu/software.html.
