PD_VERSION

Pd is a real-time graphical programming environment designed for creating live interactive computer music, and is compatible with Linux, Apple macOS, and Microsoft Windows. While Pd is copyrighted, it is available for free and can be used for any reasonable purpose. Please refer to the file PD_BASEDIR/LICENSE.txt for more information.

The Pd reference manual can be found in PD_BASEDIR/doc/1.manual/index.htm or online at http://msp.ucsd.edu/Pd_documentation/. For additional information such as guides, development, meetings, and more documentation, visit http://puredata.info. Pd's code repository is located at https://github.com/pure-data/pure-data.

The Pd mailing list archive can be accessed at https://lists.puredata.info/pipermail/pd-list.
